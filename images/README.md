## Image Optimization

Before adding images:
1. **Resize** to recommended dimensions
2. **Compress** at https://tinypng.com/
3. Keep file size under 500KB
4. Use .jpg for photos, .png for logos

## Quick Start

To see images on your site:

1. Add `hero-chikankari.jpg` to `hero/` folder
2. Add product images to `products/` folder
3. Update Hero component (see QUICK_IMAGE_SETUP.md)
4. Refresh your browser!
